# Coding-Assignments

## Repository of Coding Assignments posed to Blunav interview candidates

#### Things to do

1. You would have been mailed a link for a markdown file which will have instructions on the task at hand
2. You have 1 week to do the assignment and send it back to us
3. You send it back by uploading the code and any required visual aids to your personal github repository and reply back to us with the link. Please keep all documentation in a separate docs folder.
4. Extra marks for a video explaining your solution and approach
